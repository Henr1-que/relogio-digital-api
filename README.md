## 📌 Sobre o Projeto

Este projeto é um relógio digital que exibe as horas, minutos e segundos em tempo real, atualizado milissegundo por milissegundo. O objetivo principal foi praticar a manipulação do DOM (Document Object Model) e trabalhar com estruturas de tempo no JavaScript (`Date`).

### 🚀 Funcionalidades
* **Atualização em Tempo Real:** Mostra a hora exata do sistema sem necessidade de recarregar a página.
* **Formatação Inteligente:** Adiciona um zero à esquerda (`0`) para números menores que 10, mantendo o padrão visual (ex: `08:05:09`).
* **Design Responsivo:** Adapta-se perfeitamente a qualquer tamanho de tela (smartphones, tablets e desktops).
* **Interface Moderna:** Visual escuro (*Dark Mode*) com efeitos de sombra e neon para facilitar a leitura.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web nativas:

* **HTML5:** Estruturação dos elementos da página.
* **CSS3:** Estilização, uso de Flexbox para centralização e efeitos visuais.
* **JavaScript (ES6):** Lógica de captura das horas e atualização dinâmica da interface.

🧠 O que eu aprendi?
Uso do objeto new Date() do JavaScript para extrair horas, minutos e segundos.

Uso do método setInterval() para criar o loop de atualização a cada 1 segundo (1000ms).

Manipulação prática de elementos HTML via JavaScript utilizando document.getElementById().

Uso de variáveis CSS e flexbox para criação de layouts centralizados e limpos.
